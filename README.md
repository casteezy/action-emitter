# ng-seed
Angular 1 starter project

## Usage
1. `npm start`
2. Navigate to [http://localhost:3000/](http://localhost:3000/)

## Notes
Includes:
 - Angular 1.5.x:
  - angular
  - angular-route
  - angular-sanitize
 - Bootstrap
 - Sass
 - jQuery
 - Font Awesome
 - Typescript
